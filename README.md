# pmfront
Frontend for Primor
---------

### In repo command:

* Local overall lint test:
   in pmfront directory:
	```
	npm run lint
	```

* Local js lint test:
   in pmfront directory:
	```
	npm run js-lint
	```

* If there is some js lints, auto fix by using:
   in pmfront directory:
	```
	npm run js-lint-fix
	```

* Local css lint test:
   in pmfront directory:
	```
	npm run css-lint
	```

* If there is some js lints, auto fix by using:
   in pmfront directory:
	```
	npm run css-lint-fix
	```

* Local html lint test:
   in pmfront directory:
	```
	npm run html-lint
	```

* Install git pre-push hook:
   in pmfront directory:
	```
	./scripts/install-hooks.bash
	```
